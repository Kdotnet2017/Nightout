The purpose of this website is to help a user plan their night out. To that end, it uses the Ticketmaster API and the Google Places API to show a user a list of 25 events by location and/or keyword, purchase tickets to that event, and then show them restaurants/bars near the venue and those establishments' locations. We're currently using Pure CSS for styling.
